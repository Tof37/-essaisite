Build From sources

  • repo init -u http://github.com/CyanogenMod/android.git -b jellybean
  • repo sync -j 18
  • cd devices
  • mkdir semc
  • cd semc
  • git clone http://github.com/CMX10/android_device_semc_es209ra -b jellybean es209ra
  • cd ../../vendor
  • mkdir -p semc
  • cd semc
  • git clone http://github.com/CMX10/proprietary_es209ra -b jellybean es209ra
  • cd ../cm/
  • ./get-prebuilts
  • cd ..
  • . build/envsetup.sh
  • brunch es209ra
  • The current generated rom isn't flashable.
  • extract META-INF/com/google/android/update-binary from latest rom and replace into your ROM.
